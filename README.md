GPA Calculator
A Java console application that calculates a student’s Grade Point Average (GPA) using course grades and credit hours. The program allows users to enter their courses one by one, converts each letter grade into grade points, and calculates the final GPA.

Features
* Enter the number of courses
* Enter each course name and credit hours
* Accept letter grades from A+ to F
* Validate grade input
* Calculate total quality points
* Calculate cumulative GPA
* Display a final GPA summary

Technologies Used
* Java
* IntelliJ IDEA
* Git
* GitHub


How to Run
1. Open the project in IntelliJ IDEA.
2. Run GPACALC.java.
3. Enter the requested course information when prompted.
4. The program will calculate and display your GPA.
